## Cloudflare Workers(Hono) コーディング規約

- コンテキスト変数の参照は`c.var.*`を使う。`c.get("*")`は使わない
- DDD には準拠しない。規模に対して過剰で複雑性が増すため。
- レスポンスは必要なフィールドがあればそれを返し、なければ id のみ返す。レスポンスボリュームを減らすため。
- WebSocket の CloseEvent Code4000 番台は 3 桁以下を HTTP Status Code と対応する形とする。1000 番台などが適切であればそちらを優先して使用する。
  - 4404
    - HTTP: 404
  - 退出: 4204
  - 部屋削除: 4410
